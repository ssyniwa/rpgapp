Gemini RPG - Foundation (Version 1.0)
Google Geminiを活用した、キャラクターが成長し、記録されるPython製RPGアプリです。
AIが生成した独創的な設定を持つキャラクターたちを指揮し、敵を打ち破りましょう。

🌟 主な特徴
AIによる動的キャラクター生成: Geminiを通じて、名前、職業、スキル、物語、初期ステータスをリアルタイムに生成します。

レベルアップ・システム: 戦闘で経験値を獲得。レベルアップによりステータスが上昇し、キャラクターが成長します。

複数キャラバトル: 選択キャラとの同時戦闘が可能。ターゲット選択やスキルの使い分けが鍵となります。

オートセーブ機能: JSON形式を採用。育てたパーティーのデータは自動で保存され、いつでも再開可能です。

オブジェクト指向設計: データ（Logic）と画面（UI）を明確に分離し、拡張性の高いコードを実現。


🚀 アプリの使い方
1. Geminiでキャラクター情報を作成
Geminiで以下のプロンプトを入力

以下のJSON形式でキャラクターを作って

{
  "name": "シオン", "job": "魔剣士", "hp": 120, "mp": 50, "atk": 22,
  "description": "闇の力を宿した剣を操る孤独な騎士。",
  "image_path": "images/sion.png", 
  "skills": ["ダークスラッシュ"]
}

スキルに関する注意点
回復スキルは「ヒール」、防御スキルは「ガード」か「プロテクション」、
バフスキルは「パワー」か「エンブレム」、デバフスキルは「カース」の文字が必要。
これらの文字を含まない場合は攻撃スキルとして扱う設定のため。

またキャラクター画像の生成をGeminiで行い、ファイルimages内に保存する。

2. アプリの起動
Bash
python rpgapp13.py

3.メイン画面での入力
json形式を入力部分にコピーし、敵か味方として登録
